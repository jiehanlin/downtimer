# 倒计时器 (DownTimer)

一个基于Flutter开发的跨平台桌面倒计时应用，支持Windows和Linux系统，兼容x86和arm64架构。

## 功能特性

- ✅ **跨平台支持**: 支持Windows和Linux系统，兼容x86/arm64架构
- ✅ **LED数字显示**: 模拟真实LED显示屏效果，显示分钟和秒数
- ✅ **窗口管理**: 支持窗口最小化（仅显示倒计时）、双击切换窗口模式
- ✅ **时间设置**: 提供快捷时间设置（15/30/45/60分钟）和分钟级微调功能
- ✅ **窗口控制**: 窗口置顶显示、可拖动调整位置、可调整大小
- ✅ **自适应布局**: 倒计时字体大小随窗口尺寸自动调整
- ✅ **计时控制**: 开始/暂停/重置计时器功能

## 界面预览

### 正常窗口模式
- 标题栏：显示应用名称，支持拖拽移动窗口
- LED显示区域：大型LED数字显示屏，显示"MM:SS"格式
- 控制面板：时间设置按钮组和控制按钮

### 精简模式（最小化窗口）
- 仅保留LED显示区块，背景透明化处理
- 双击LED区域可切换回正常窗口模式

## 技术栈

- **框架**: Flutter 3.0+
- **语言**: Dart
- **状态管理**: Provider
- **窗口管理**: window_manager
- **本地存储**: shared_preferences

## 快速开始

### 环境要求

- Flutter SDK 3.0+
- Dart SDK 3.0+
- 桌面开发环境配置

### 安装依赖

```bash
flutter pub get
```

### 运行应用

```bash
# Windows
flutter run -d windows

# Linux
flutter run -d linux
```

### 构建发布版本

```bash
# Windows
flutter build windows

# Linux
flutter build linux
```

## 使用说明

### 基本操作

1. **设置时间**: 
   - 点击快捷时间按钮（15/30/45/60分钟）
   - 使用+/-按钮以分钟为单位微调

2. **控制计时**:
   - 点击"开始"按钮开始倒计时
   - 点击"暂停"按钮暂停计时
   - 点击"重置"按钮重置到初始时间

3. **窗口操作**:
   - 拖拽标题栏移动窗口
   - 双击LED显示区域切换窗口模式
   - 点击最小化按钮进入精简模式

### 快捷键

- **双击LED区域**: 切换窗口模式
- **鼠标滚轮**: 快速调整时间（待实现）
- **右键菜单**: 快捷设置选项（待实现）
